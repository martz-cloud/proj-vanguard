# Setup

1. Clone the repository.
2. Review prerequisites in the README.
3. Follow the configuration steps below.

```bash
# example
git clone https://github.com/<ORG>/<REPO_NAME>.git
cd <REPO_NAME>
```

## Configuration
- Fill in placeholders like `<TENANT_ID>`, `<CLIENT_ID>`, `<ORG_NAME>`, `<APP_NAME>`, `<ENV>` as applicable.
